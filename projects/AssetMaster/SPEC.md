# SPECIFICATION: AssetMaster (全方位資產管理系統)

## 🎯 Goal
開發一個支援美股、日股、台股及加密貨幣的投資管理系統。重點在於直觀的「資產狀況」儀表板，讓使用者能一目了然整體的配置與損益。

## 📋 Requirements
- **多市場支援**: 需處理不同幣別 (USD, JPY, TWD, USDT/BTC)。
- **即時/延遲報價**: 整合 API 獲取主要市場的現價。
- **交易紀錄 (Ledger)**: 支援「買入」、「賣出」、「股息/配息」紀錄。
- **資產儀表板 (Visualization)**: 圓餅圖（配置佔比）與 趨勢圖（損益）。
- **匯率轉換**: 自動根據當前匯率轉換為單一基準貨幣 (Base Currency)。

## 🛠 Technical Stack
- **平台**: Web-based (可選 Next.js + TailwindCSS) 或 Google Sheets (極簡版)。
- **數據源**: Yahoo Finance API (股票), Binance API (加密貨幣), 台灣證交所 API。
- **資料庫**: SQLite (本地) 或 Supabase (雲端)。

## 📐 Architecture / Flow
1. **輸入層**: 使用者輸入交易詳情。
2. **處理層**: 背景任務抓取最新價位並計算成本/損益。
3. **展示層**: DASHBOARD 提供視覺化資產分布圖。
