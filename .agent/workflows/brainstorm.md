## 🧬 Brainstorming Workflow

Follow these steps to capture a new idea:

1.  **Target Directory**: Ensure you are in the `ideas/` folder.
2.  **Idea Initiation**:
    -   Ask the User: "給這個靈感一個簡短的名稱 (slug)？"
    -   Create file `ideas/{slug}.md`.
3.  **Capture Phase**:
    -   Ask the User: "請隨意描述您的想法、痛點或願景。"
    -   Write the raw thoughts to the file under a `# 💭 Raw Thoughts` section.
4.  **Expansion Phase (Creative Rule Applied)**:
    -   Apply `CORE_CREATIVE.md` logic.
    -   Suggest 3 potential directions or "What If" scenarios.
    -   Append these to the file under `# 🚀 Potential Directions`.
5.  **Conclusion**:
    -   Notify User: "💡 靈感已存檔。隨時可以進行深度發想，或準備好後執行 `/confirm-requirement` 轉移至需求分析。"
