## 📖 這是什麼？ (The Core Mission)
石虎 AgentOS 是一套設計用來處理「AI Session 遺失」與「跨 Agent 協作斷層」的作業系統。它透過 **物理記憶層 (LAMP)** 與 **邏輯/數據分離架構**，確保無論你換了哪一個 AI（Antigravity, Cursor, Gemini），系統的「靈魂」都能完美繼承。

### 🌟 核心價值：
*   **不朽記憶**：每一輪對話都會即時備份至磁碟，不依賴雲端 Chat History。
*   **戰位繼承**：透過角色繼承樹，讓 AI 瞬間進入「鳴風、織圖、虎掌、銳爪」特定戰位。
*   **真相中心**：數據與代碼物理分離，支援一鍵切換開發領域（公司/個人/機密）。

## 🚀 快速開始 (Quick Start)

### 1. 一鍵入籍 (Installation)
在你的 Linux 環境中執行以下指令進行初始化：
```bash
./scripts/bootstrap.py  # 自動修復數據橋接與環境
./install.sh            # 初始化核心維生服務
```

### 2. 立即體感 (Hello AgentOS)
我們準備了一個範例專案，讓你快速測試 Agent 的「靈魂注入」能力：
```bash
./bin/status            # 查看目前專案清單
./bin/run-workflow      # 啟動 hello-agentos 體驗流程
```

## 🐾 蜂群戰位 (Role-Tree Matrix)
本 OS 依據開發週期定義了四大父類角色，AI 會根據你的需求自動「降臨」：

| 戰位名稱 | 核心使命 | 適合情境 |
| :--- | :--- | :--- |
| **🌬️ 鳴風 (Whisperer)** | 抓取 Vibe / 靈感捕捉 | **許願池模式**：只有模糊想法時。 |
| **🧶 織圖 (Weaver)** | 規格制定 / SDD 顧問 | **規訓模式**：要將想法轉化為技術文件時。 |
| **🐾 虎掌 (The Paw)** | 代碼肉搏 / 環境修復 | **實作模式**：進入具體編程與除錯時。 |
| **🦅 銳爪 (The Claw)** | 安全審計 / 破壞測試 | **審計模式**：執行最終驗證與安全掃描時。 |
