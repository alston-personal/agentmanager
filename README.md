# 🐱 石虎 AgentOS (LeopardCat AgentOS) - 功能全盤點 2.0

## 🌌 系統憲法：數據與邏輯的物理分離 (Logic/Data Separation)
石虎 AgentOS 不是一個單純的代碼庫，它是 AI 的 **自癒運作系統**。
它將「具備揮發性的對話」轉化為「永恆的物理紀錄」，確保 Agent 遺失對話時，靈魂依然留在數據層。

---

## 🛠️ 核心功能矩陣 (Feature Matrix)

### 1. 🗄️ 真相中心 (Data Layer)
*   **物理架構**：所有進度與記憶均存在於 `/home/ubuntu/agent-data`。
*   **域名切換 (`/switch-brain`)**：一鍵切換 `Work / Personal / Secret` 數據集。
*   **Symlink Bridge**：全自動初始化各專案的記憶橋樑。

### 2. 📡 蜂群維生系統 (Vital Support)
*   **禪定貓 (Zen Meditation)**：每日凌晨 00:00 執行全系統自診斷與健康報告生成。
*   **守夜貓 (Night Watcher)**：15 分鐘一次的自動修補，清理垃圾文件、確保 Docker 通道暢通。
*   **貓墨同步 (Cat-Ink Syncer)**：[New] 每 60 秒一次的對話備份，防止 Antigravity Server 遺失對話。

### 3. 🧠 跨代理記憶協議 (LAMP Protocol)
*   **遺產繼承 (`handover.py`)**：自動生成「接班手冊」，讓不同時間進場的 Agent 瞬間接手。
*   **共感脈搏 (.agent/pulse/)**：跨工具存取（Antigravity, Cursor, Gemini...），不依賴特定工具的 Chat History。

### 4. 🚀 快速戰術操作 (Workflows)
*   **`/status`**：儀表板視覺化監控，一覽 20+ 專案脈搏。
*   **`/report`**：手動強迫同步目前的進度到數據中心。
*   **`/brainstorm`**：開啟具備 Vibe Coding 靈性的原始需求捕捉。

## 🐾 戰位佈署 (Role Matrix)
本作業系統依開發週期將 Agent 職能劃分為：
*   **鳴風 (Whisperer)**：發想/抓取 Vibe。
*   **織圖 (Weaver)**：規訓/產出嚴謹規格 (SDD)。
*   **虎掌 (The Paw)**：肉搏/執行純淨開發。
*   **銳爪 (The Claw)**：審計/破壞性測試。
